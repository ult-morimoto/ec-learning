✅ 機能要件（機能一覧）
1. 商品一覧表示機能
   固定の商品データを配列などで保持（外部APIなどは使用しない）

商品名、価格、画像、説明を表示

2. 商品のカート追加機能
   各商品に「カートに追加」ボタンを設置

クリックするとローカルストレージに追加

同じ商品は数量が加算されるようにする

3. カート表示機能
   カートの中身をモーダルまたは別ページで確認可能

商品名、数量、合計金額を表示

商品の削除・数量変更機能

4. 購入手続き（簡易）
   「購入」ボタンで確認画面を出す（購入処理は行わない）

購入内容をローカルストレージから削除して初期状態に戻す

5. データの永続化
   ローカルストレージでカートの状態を保存し、リロードしても保持されるようにする

6. UI/UX
   基本的なレスポンシブ対応（スマホ/PC）

エラーハンドリング（在庫0などの例外がある場合）

📦 非機能要件
使用言語：HTML, CSS, JavaScript（Vanilla JS）

ライブラリ：なし（可能であればDOM操作を中心に）

バックエンド：不使用

データ保存先：localStorage または sessionStorage
